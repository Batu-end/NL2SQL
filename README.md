# NL2SQL
WIP chatbot that will perform NL to SQL queries

# How to run
- create venv
- `pip install -r requirements.txt`
- go to directory and do `uvicorn main:app --reload`
- go to `https://127.0.0.1:8000/docs` and press "try it out"
- replace "string" after "question:" with actual question and execute
- scroll down and check result
